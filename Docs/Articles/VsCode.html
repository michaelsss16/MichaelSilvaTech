Oque é o vscode?

Antes de tudo, vamos entender o que é o vscode.
O Visual Studio Code é um editor de código-fonte leve, mas poderoso, que é executado na área de trabalho e está disponível para Windows, macOS e Linux. Fornece suporte interno para JavaScript, TypeScript e Node.

Quais linguagens o vscode aceita?

O VSCode atende uma quantidade enorme de projetos (ASP . NET, Node. js) e oferece suporte a mais de 30 linguagens de programação como JavaScript, C#, C++, PHP, Java, HTML, R, CSS, SQL, Markdown, TypeScript, LESS, SASS, JSON, XML e Python assim como muitos outros formatos de arquivos comuns.

Como trabalhar com o vscode?

Abra seu VS Code e na tela inicial abra o Command Palette (você consegue acessar ele usando o atalho Cmd+Shift+P) e busque a opção “Shell Command: Install 'code' command in PATH“. Pronto, você já consegue acessar o vscode pelo comando code no terminal.

Como instalar

Para Windows:
- A instalação no Windows é relativamente simples, basta acessar o site oficial do Visual Studio Code e fazer o Download da sua última versão, clicando no botão "Download for Windows".
Para Linux:
- A instalação no Linux é relativamente simples, basta acessar o site oficial do Visual Studio Code e fazer o Download da sua última versão, clicando no botão "Download for Linux". Como resultado do Download você terá um pacote . deb na pasta Downloads.
